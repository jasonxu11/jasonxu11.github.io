新的站点、新的起点

发表于 2016-04-25   |   分类于 NodeJs   |     |   本文总阅读量 次


经过两晚的倒腾，终于完成了hexo基于github的搭建，之前一直在CSDN上写博客，在工作的前期也坚持了一段时间，后来是越发的少了，猛然间看到各位年轻于我们的小伙伴们突军而起，倍感鸭梨。其实每个coder在工作中无论是从事项目还是研发，都有不少值得分享的经验，基于实战都是干货，写博客的目的，一是在与分享，我们每个IT从业者都应该乐于分享，我们应该庆幸我们生活在互联网如此发达的今天，我们可以从网上找到为我们解惑的答案，我们都从statckoverflow、CSDN、ITEYE、CNBlog等诸多大牛和小神那取得过真经，也从github、OSchina练得葵花宝典、寻的存在感和价值，我们的知识来源于社会，那么就不要吝啬吧，分享对自己也是一种成长。二是在于经验备份，哈好记性不如烂笔头，越是年纪大，越是感觉虚…，每次学过、经历过的技术疑难点，当时是搞的清楚了，过了一段时间，总得努力回忆下，此处应该会有嘘声…。哈不多扯了，To me. this is a new starting point，come on！

如果有使用hexo搭建博客存在问题的小伙伴大家可以多多交流

坚持原创技术分享，您的支持将鼓励我继续创作！
